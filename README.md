# Vault-Ed: Your Digital Financial Twin

A premium, elegant personal finance web application designed for high school students to learn financial management through interactive features.

## Features

### 1. **Dynamic Wallet** 💳
- Three sub-wallets: Ready to Spend, Savings, and Emergency
- Visual balance display with percentage breakdown
- Automatic deduction from correct wallet on transactions
- Easy wallet transfer between categories

### 2. **Transaction Logging & Smart Categorizer** 📝
- Quick transaction entry with category selection
- Pre-defined categories: Canteen, Transport, Gaming, Entertainment, Shopping, Education, Health, Other
- Transaction history with filtering by category and month
- Edit and delete transactions with automatic refunds

### 3. **Monthly Infographic Report (Spotify Wrapped Style)** 📈
- Beautiful monthly summary with spending insights
- Category breakdown visualization
- Savings rate calculation
- Financial tips based on spending patterns
- Shareable report design

### 4. **Milestone Map - Savings Goal Tracker** 🎯
- Create named savings goals (e.g., "Buy a Phone")
- Visual progress bars with percentage completion
- Target amount and deadline tracking
- Add funds to goals directly from savings
- Goal completion celebration

### 5. **Time Machine Simulator** ⏰
- **Compound Interest Calculator**: Project savings growth over 10+ years
- **Daily Habit Cost Calculator**: Calculate long-term cost of daily habits (e.g., daily coffee over a year)
- Interactive input forms with real-time calculations
- Side-by-side comparisons for decision-making

### 6. **Virtual Interest Rate System** 🏦
- Automatic 2% annual interest on Savings wallet
- Monthly bonus calculation
- Progress tracker showing days until next bonus
- Educational tool to teach how bank interest works

### 7. **Budgeting Challenge - Weekly Leaderboard** 🏆
- Invite friends to weekly spending challenges
- Real-time leaderboard showing lowest spenders
- Weekly reset (Monday to Sunday)
- Competitive motivation to save more

### 8. **Predictive AI - Runout Warning** ⚠️
- Analyzes first-week spending patterns
- Calculates daily average spend
- Predicts estimated runout date
- Proactive warnings before money runs out

### 9. **Teacher-Parent Link** 👨‍👩‍👧
- Parents can assign financial tasks/challenges
- Students complete tasks to unlock real-world rewards
- Task tracking and completion history
- Educational framework for financial responsibility

### 10. **Responsive Dual-Mode UI** 📱💻
- **Mobile-first design**: Quick expense entry, receipt scanning ready
- **Desktop dashboard**: Full analytics, detailed graphs, printable reports
- Responsive breakpoints for all devices
- Dark mode support
- Premium, polished visual design

## Getting Started

### Installation

1. Extract the ZIP file to your desired location
2. Open `index.html` in a modern web browser
3. Start managing your finances!

### Usage

1. **Add Funds**: Use the Wallet page to add initial funds to your wallets
2. **Log Transactions**: Record expenses in the Transactions page
3. **Set Goals**: Create savings goals in the Savings Goals page
4. **View Reports**: Generate monthly reports to see spending insights
5. **Use Simulator**: Calculate compound interest and habit costs
6. **Join Challenges**: Invite friends for weekly budgeting challenges
7. **Complete Tasks**: If a parent assigns tasks, complete them to unlock rewards

## Data Storage

All data is stored locally in your browser using LocalStorage. Your financial information never leaves your device.

### Backup & Export

- **Export as JSON**: Backup all your data in JSON format
- **Export as CSV**: Export transactions for spreadsheet analysis
- **Reset Data**: Clear all data and start fresh
